# Play Store Submission Checklist

- Upload keystore is created and stored safely outside the repo
- `keystore.properties` contains the real local signing values
- `build_play.bat` completes successfully
- `app\build\outputs\bundle\release\app-release.aab` exists
- App icon is `512 x 512`
- Feature graphic is `1024 x 500`
- Minimum 2 phone screenshots are ready
- Privacy policy URL is live
- Store listing copy is approved
- Data safety form is completed
- Content rating questionnaire is completed
- App access instructions are prepared if login or OTP is needed for review
- Release notes are prepared
- Final production URL is verified and working on HTTPS if available
